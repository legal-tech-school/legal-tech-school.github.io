How to get and download/save an email attachment in Python 3

Since I couldn't find any specific answer to what seemed like a very common problem - logging in to an email server and downloading a specific email attachment in Python 3 - anywhere on the Information Super-Highway, I thought I'd post my solution here.

import imaplib
# Log in to the mail server with imaplib
mail = imaplib.IMAP4_SSL('mailserver')
mail.login('user_id','password')
mail.select("INBOX")
result, data = mail.uid('search',None, "ALL")


# this identifies the latest email in the inbox. To get the next-to-latest change -1 to -2 and so forth
email_uid = data[0].split()[-1]
result, data = mail.uid('fetch', email_uid, '(RFC822)')
raw_email = data[0][1].decode('utf-8')
email_message = email.message_from_string(raw_email)


# let's find out about our email -
print(email_message['To'])
print(email_message['From'])
print(email_message.items())
# get the payload - basically the parts of the email.
payload = email_message.get_payload()
print(payload) # debug with print statements, like a pro


# 0 is typically the body of the email, 1 is the attachment,
# assuming you only have one attachment that you want.
attachment = payload[1]
your_filename = 'your-filename.whatever'
open(your_filename, 'wb').write(attachment.get_payload(decode=True))
