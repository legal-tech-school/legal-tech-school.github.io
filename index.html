WHY PYTHON IS A GREAT LANGUAGE FOR LAWYERS:

It's approachable.

Python code is easy / easier to read and grasp what's happening in a program than many other languages.

It's easy to run.

With Python you don't have to worry about compiling or messing up the order of your punch cards. In fact, you might already have a version of python installed on your computer.

It's a general-purpose language.

There's a lot of things that you can do with Python, from machine learning and data science to creating websites (this website is created using Python, for example).

Many free resources.

Python has been around a while, and, because it's a great language, there are many quality tutorials that also happen to be free. Check out the resources page for a list.

About Me:

I'm a founder of Legal Analytics Corporation and 'Jack of all Trades' at MyCourtCase.org, a website that gives average people the ability to see the results of hundreds of thousands of past criminal cases like theirs, and make an informed decision about what to do in their case.

Most recently I deployed a web app that lets people in Hillsborough County easily find out when they need to be in court, that's called Court Date Search. Tampa developer Mike Turtora collaborated on this project.
